‚òÅÔ∏è Setting Up filen.io Sync

‚ö†Ô∏è Important: By default, FlexiBase uses local browser storage. Your data is stored on your device only. To enable cloud sync, follow these steps.

Step 1: Get filen.io Credentials

  1. Sign up at filen.io
  2. Go to your account settings
  3. Generate an API key
  4. Note your folder ID where you want to store data

Step 2: Configure Storage

Copy config.template.js to config.js and fill in your credentials:

const CONFIG = { filen: { apiKey: 'your-api-key-here', folderId: 'your-folder-id-here', autoSyncInterval: 300000 // 5 minutes } };

Step 3: Update Storage Module

In storage.js, implement the filen.io API methods. Detailed instructions are in the README.

‚úì Pro Tip: Test with local storage first, then add cloud sync once you're comfortable with the app.
